## 2. Agent 层关键词

代码位置：
- [src/agent/planning/action-planner.ts](/D:/Music-Skill/src/agent/planning/action-planner.ts)

这些关键词不会直接走基础 skill 路由，而是先经过 `MusicAgentService` 决策。

### 2.1 偏好播放

触发词：
- `按我的喜好播放`
- `按我的喜好来点`
- `来点我喜欢的`
- `放点我喜欢的`

当前行为：
- 走 agent 推荐链路
- 结合收藏、偏好记忆、行为记忆、场景记忆选歌

### 2.2 写代码场景

触发词：
- `来点适合写代码的`
- `来点写代码听的`
- `写代码的时候听什么`
- `给我放点适合编程的`

当前行为：
- 激活 `coding` 场景
- 走场景推荐链路

### 2.3 安静场景

触发词：
- `来点安静的`
- `放点安静的`
- `想听安静一点的`
- `来点轻一点的`

当前行为：
- 激活 `calm` 场景
- 走场景推荐链路

### 2.4 记忆偏好

触发词：
- `记住我喜欢...`
- `我喜欢这种...`
- `以后多放...`

当前行为：
- 写入长期偏好记忆

示例：
- `记住我喜欢周杰伦`
- `记住我喜欢这种安静的人声`

### 2.5 负向偏好

触发词：
- `以后别放...`
- `不要再放...`
- `我不喜欢这种...`

当前行为：
- 写入负向偏好记忆

示例：
- `以后别放这种太吵的`

代码位置：

- `音乐控制，收藏这首歌`
- `音乐控制，按我的喜好播放`

## 5. 建议修改顺序

如果你后面要统一改关键词，建议按这个顺序：

1. 先改 `intent-router.ts`
2. 再改 `action-planner.ts`
3. 再改 `slot-extractor.ts`
4. 最后改 `start-voice.ps1` 里的唤醒词
